# jQuery Maps

A jQuery library that adds Google Maps to DOM elements using HTML data attributes.

## Usage

```html
<div class='map' data-pickup="Heathrow Terminal 1" data-destination="London Waterloo"></div>
```

```javascript
$(".map").journey();
```
